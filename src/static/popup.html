Coplit to Github

made with 🔥 by Donghun Seol, Duhoon Kang

https://github.com/Coplit-To-Github/CoplitToGithub

코플릿 코드를 깃헙에 자동으로 커밋해주는 크롬 익스텐션 입니다.

자신의 깃헙에 coplit private 리포를 만들고, 그곳에 코드를 업로드 합니다.

⚠️ Coplit to Github 익스텐션으로 생성한 리포와 커밋를 public으로 변경하는 것은 엄격히 금지됩니다. ⚠️

깃허브 api를 이용하므로, 깃헙 아이디와 accessToken이 필요합니다.

아래에 깃헙 아이디와 accessToken을 입력하고 저장버튼을 눌러주세요

모든 요청과 데이터는 로컬환경에서 처리됩니다.

👇 이곳에 정보를 입력하세요 👇
